# Linux Media Fix Tools

Tools and documentation for diagnosing and resolving common Linux audio, video, and codec issues.

## Focus Areas

- Missing or broken codecs and playback issues
- PipeWire or PulseAudio configuration and reset procedures
- GPU acceleration checks
- Player configuration guidance for common media players
